About Us


Who are we?


D.A.V. Authentications is a acronym of Deepanjali, Anuj, Vanshaj, the developers of DAV Authentications. This website is a result of a project given to the trio for finding a solution to the problem of fake certification. The engineers have its expertise in Web Development and programming languages to tackle this issue via this web based solution by the name of DAV Authentications.

What is DAV Authentications?


DAV Authentication is a modern web based solution to plagiarism or faking of certifications or documents in a system. Fake certificates and documents running in the corporate or government offices tend to major losses to the companies or to the country as people working in the posts are not qualified enough to perform the task, hence resulting in failures and losses. Plus such cases of fraud lead to law and order crisis in the country. DAV Authentication is a web based solution to the above mentioned problem which used modern technologies to verify the originality of any document via a UUID code (Universal Unique Identity) which is unique to any document and the UUID code can be uplinked to this website, one can verify the originality of the document and also access all the supplementary details relative to the document. Ones a UUID is uploaded to our website, the algorithm of the system will search our database of all existing files and give the user all required details. All the data in the server is well encrypted and all data is safe i.e. only authorized personal will be able to access the data.
